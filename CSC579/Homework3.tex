\section*{Question 2}
The Gauss-Siedel was computed on the Harwell Boeing matrix using the following
matlab code 
\begin{lstlisting}[frame=single] 
 function pi = HW3_gaussSeidel(pi, V, J, I)
    for s = 2 : size(I, 2)  
        sum = 0;
        for i = I(s-1):I(s)-1
            sum = sum + pi(J(i))*V(i);
        end  
        pi(s-1) = sum;
    end
end 
\end{lstlisting}
Using the values \\
$ V = \left(\begin{array}{ccccccc}

\end{array}\right)$\\

\end{array}\right)$\\
